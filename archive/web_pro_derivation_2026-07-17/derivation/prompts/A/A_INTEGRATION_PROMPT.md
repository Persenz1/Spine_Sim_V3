## 1. 研究背景与系统位置

本项目研究钩爪式爬壁机器人在红砖、混凝土和不同目数砂纸等硬质粗糙墙面上的爪刺啮合。机器人使用有限半径球形尖端的高碳钢针刺，在法向主动推力下沿爪单元局部正 x 方向连续搜索和拖拽。完整求解器必须从表面查询开始，依次形成单刺状态演化、阵列活动接触集与载荷共享、四单元十字对爪整体平衡。

物理依赖链固定为：

表面查询 → 单刺算子 A → 阵列爪单元算子 B → 十字对爪算子 C。

当前任务只集成大模块 A。A1 已建立表面数据与质量、有限球尖可达性、合法球冠接触、支持点/法向、复合针体禁止碰撞和几何事件；A2 已建立单边接触、三维 Coulomb 摩擦、最大耗散、刚性/梁针/独立轴向弹簧柔顺、恒法向主动推力与切向位移的混合边界、非光滑平衡和事件定位；A3 已建立真实滑移提交、三维接触迁移、局部材料容量与尺度桥接、针体强度、不可逆损伤、释放、继续搜索、再挂接、多峰轨迹和事务式单刺试探接口。

三个子模块已经接受，但顺序完成不等于大模块已经集成。本轮必须把它们重构为一个符号唯一、状态闭合、事件可排序、功方向一致、没有重复柔顺或重复失效、可执行单步推进、可由 B 层稳定调用的正式单刺模型。不得只拼接、压缩或摘要现有章节。

与集成直接相关的固定工程背景包括：

- 墙面名义平面为全局 X-Y；正 Z 从墙面指向背板和自由空间。
- 单元局部正 x 从头部指向根部，也是搜索和拖拽方向；局部 y 与正 Z 组成右手系。
- 只有局部球形针尖允许承载；锥段、针杆和安装座只参与禁止碰撞。
- 针尖半径为 50 或 100 μm，针径为 0.6 或 0.8 mm；固定角单针球心露出长度为 4 mm。
- needle_bending 的 off/on 是正式开关；关闭弯曲不等于关闭针体强度检查。
- 独立轴向弹簧刚度范围为 100–2000 N/m，无预压、只压缩、不承拉，压缩范围为 0–4 mm，到 4 mm 后进入刚性硬限位。
- 单刺独立验证工况的法向主动推力固定为 0.5 N；恒定的是执行器推力，不是任一局部法向反力。
- 阵列爪单元层的法向主动推力为每单元 0.5–2 N，必须由 B 层共同平衡分配；不得把单刺独立验证的 0.5 N 复制成阵列内每根针的强制法向载荷。
- 单刺与阵列拖拽速度为 1 mm/s，仅用于位移—时间映射；忽略惯性，连续拖拽行程上限为 100 mm，脱离后不重置路径。
- 一次连续仿真内保留轻量不可逆损伤；新独立试验或新表面样本从无损状态开始；原始表面几何不因轻量损伤被重写。
- 首版排除显式裂纹扩展、碎屑动力学、连续切削有限元、地形重网格化、针尖磨损、导轨/框架柔性和惯性动力学。
- 摩擦、局部接触柔顺、目标材料强度/断裂、针体牌号/强度、损伤核、搜索控制策略和数值容差仍是参数、模型选择或待标定项。

## 2. 本轮唯一目标

输出正式 A 单刺算子及冻结的 A→B 合同，使同一个 A 对象能够：

1. 在独立单刺验证模式中，按固定 0.5 N 法向主动推力和切向位移推进完成 100 mm 连续轨迹；
2. 在 B 层嵌入模式中，接受 B 给定的针基座位姿/共同增量与不可变历史快照，返回针对背板的试探 wrench、状态、事件、切线/割线、损伤试探更新和质量诊断，而不在每根针内部重复施加 0.5 N 法向主动推力；
3. 只在调用方接受全局步骤后提交滑移、损伤、路径和事件历史，并可完整回滚；
4. 保持 A1 几何、A2 接触/柔顺、A3 滑移/材料/损伤为单向依赖，不让下游重新实现低层机理。

必须明确区分两个层次：

- intrinsic_single_spine_kernel：给定基座位姿/增量后求单刺几何、接触、结构、材料和事件的本征响应；这是 B 层调用入口。
- standalone_single_spine_driver：在本征核外施加单刺 0.5 N 法向主动推力、搜索控制和 100 mm 独立试验路径；只用于 A 层验证与单刺实验对比。

若不做这一分层，A2/A3 中的单刺恒推力残量会与未来 B 层的每单元总法向推力重复施加，A→B 合同即不合格。

## 5. 必须完成的一致性集成

### 5.1 规范对象和状态所有权

给出一张规范对象表，至少列出：

- SurfaceRealization 与 A1QueryHandle；
- composite needle geometry 与针级参数包；

### 5.2 唯一坐标、符号、单位和参考点

建立正式符号表，消除同义词和隐式换号。至少明确：

- 全局基、单元局部基、针局部基和每个接触局部基之间的变换；
- 正 Z、正 x、针轴、间隙、法向力、切向力、墙对针与针对墙的作用—反作用；

- wrench 的声明参考点，参考点变换公式和力矩单位 N·mm；
- 长度 mm、力 N、时间 s、应力 MPa、刚度 N/mm、输入弹簧 N/m 的转换、角度 rad；
- 独立单刺抓附反力、执行器驱动力和 B 层针级反力之间的符号映射。

不得只写“符号与原文一致”。A_TO_B_CONTRACT 必须能让 B 在不阅读 A 内部推导的情况下正确装配力与力矩。

### 5.3 功、储能和耗散

给出统一离散功平衡和每一项的正方向。必须检查：

- 切向位移控制反力的外功；
- 法向主动推力在独立单刺 driver 中的外功；
- B 嵌入模式下由基座 wrench 与基座增量定义的功共轭，不再在针内重复计入单刺法向执行器功；
- 梁储能、轴向弹簧储能、可选局部接触储能；
- Coulomb 摩擦耗散和材料损伤耗散；
- 接触释放后的可恢复能返回项；
- 理想刚性约束、粘着约束和硬限位在约束方向零功；
- 数值残差不被解释为物理耗散。

要求坐标变换和参考点变换不改变总功；要求试探/回滚不累计耗散。

### 5.4 方程与约束的唯一装配

把正式单步问题写成可实现的残量/互补块，至少包含：

- A1 合法球冠间隙、支持图表和非承载体硬不等式；
- A2 Signorini–Coulomb 二阶锥互补和最大耗散；
- 梁兼容、轴向弹簧原长/压缩/硬限位分支；
- 接触迁移图表、滚动与滑移区分；
- 局部材料容量、软化互补、针体强度上限；
- 独立单刺 driver 的 0.5 N 法向混合边界；
- B 嵌入核的规定基座运动边界，其中不施加每针 0.5 N；
- 每个模式下的未知量、方程数/约束、硬不等式和后处理反力。

若 A2/A3 现有 standalone 残量和 B trial 接口不能直接兼容，必须通过“本征核 + driver 包装层”重构，而不是删除任何一方。

### 5.5 重复柔顺和重复失效审计

逐项给出“物理位置—状态量—力—位移—能量—开关—不得重复项”矩阵：

- 球尖—表面局部接触柔顺；
- 露出针梁；
- 针轴向独立弹簧；
- 背板刚体约束和硬限位；
- 夹具/根部柔顺若未来启用；
- 材料损伤容量缩减。

必须明确：

- 轴向弹簧压缩与梁轴向/弯曲变形是串联但不同物理部件；
- 局部接触压缩不能用总系统斜率反演后再与梁/弹簧重复相加；
- 材料损伤缩减强度/容量，不得同时无依据降低摩擦系数、梁刚度或弹簧刚度；
- 针体屈服/断裂和墙面材料失效分别检查，不压缩成无解释的最小标量；
- 复合针体碰撞不是材料失效，也不是新增承载接触。

### 5.6 规范状态机

整合 A1/A2/A3 重复枚举为：

1. 一个互斥主机械状态；
2. 接触点运动子状态；
3. 弹簧子状态；
4. 表面材料子状态；
5. 针体强度子状态；
6. 质量/求解状态；
7. 瞬时事件记录。

对每个状态写明进入条件、保持不变量、允许转移、退出事件和历史提交。特别处理：

- 完全分离且受非零恒力时不存在静态平衡；standalone driver 采用受控几何搜索，embedded kernel 只返回 open response/事件，不伪造平衡；
- TIP_ZERO_LOAD 到预载建立；
- 锥边界不自动等于真实滑移；
- 支持点迁移不自动等于材料滑移；
- 材料启动/软化不自动等于接触释放；
- 释放后梁/弹簧可逆回位但损伤、路径和时间不回退；
- 再挂接不恢复完整材料容量。

### 5.7 事件优先级和并发竞争

给出唯一事件表和确定性处理规则。固定致命优先级必须为：

1. OUT_OF_DOMAIN 或 GEOMETRY_UNCERTAIN：输入不足，停止物理排序；
2. 锥段、针杆或安装座碰撞：纯球尖承载模型无效。

其他事件先定位到共同位置，再做并发分支竞争，不得用简单 if/else 静默覆盖：

- 接触建立/释放；
- 支持图表切换或迁移退化；
- 摩擦边界/真实滑移；
- 弹簧原长/硬限位；
- 材料起始/软化/完全失效；
- 针体屈服/断裂；
- 平衡退化、物理无解或数值未收敛。

必须区分事件函数量纲、数值缩放、并发容差、事件一侧一致性、终止事件与可继续事件。针体断裂/线弹性屈服边界可终止首版分支，但不能删除同位置的其他事件记录。

### 5.8 可执行单步算法

给出从 accepted state n 到 accepted state n+1 的完整伪代码，至少包含：

- 不可逆试探更新后完整平衡重求解；
- 分支接受顺序、功/耗散和质量检查；
- 原子 commit 或完整 rollback；
- 剩余外部增量循环、零长度事件循环保护和最小步失败分类；

## 6. A→B 合同必须冻结的内容

A_TO_B_CONTRACT 必须是 B 可独立使用的稳定公共合同，同时逐字嵌入 A_INTEGRATED_MODEL 的“公共 A→B 接口”章节。两处合同内容必须逐字一致，不允许一个是摘要、另一个是完整版。

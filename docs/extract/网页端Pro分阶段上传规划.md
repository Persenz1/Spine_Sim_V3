# 网页端 Pro 分阶段上传规划

## 使用方法

- P 编号与 `机理提取/` 下同编号目录对应。例如 P01 使用 `机理提取/01_frictional_engagement/evidence_card.md`。
- “完整提取”上传该论文完整的 `evidence_card.md` 及任务相关图片；“相关片段”只上传相关机理段落和图片；“证据卡”只上传结论、适用范围和限制。
- 每一批使用新的 Pro 对话；后一批同时上传表中列出的前置结果。

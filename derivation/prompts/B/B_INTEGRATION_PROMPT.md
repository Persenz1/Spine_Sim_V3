## 1. 研究背景与系统位置

本项目研究钩爪式爬壁机器人在红砖、混凝土和不同目数砂纸等硬质粗糙墙面上的爪刺啮合。机器人爪单元由规则矩形多刺阵列组成，每根高碳钢针具有有限半径球形针尖；阵列安装在绝对刚性背板上，可采用刚性安装或彼此独立、与针轴共线的单边压缩弹簧安装。爪单元在恒定法向主动推力下沿局部正 x 连续搜索和拖拽，必须真实产生部分接触、载荷不均、失效重分配和再挂接，而不能用“有效刺数×平均单刺力”替代共同平衡。

完整物理依赖链固定为：

表面查询 → 单刺算子 A → 阵列爪单元算子 B → 十字对爪算子 C。

当前任务只集成大模块 B。B1 已建立规则阵列几何、针级身份、固定角/线性角度梯度、实际露出长度、球心共面、共同刚性背板运动、刚性/独立弹簧拓扑、方向邻接与 A 单刺 embedded 请求；B2 已建立给定共同切向位置与每单元恒定法向主动推力时的共同法向平衡、正交针级状态、活动集合、刚性集合值 graph、逐针 wrench 装配、切线/割线、共同事件减步和失败分类；B3 已建立失效后全阵列重平衡、自动载荷重分配、同时事件与级联、共享 DamageStore 协调、原子事务、释放/回弹/继续搜索/再挂接、100 mm 连续历史和历史相关单元能力候选接口。

三个子模块已经接受，但顺序完成不等于大模块已经集成。本轮必须把它们重构为一个状态唯一、方程闭合、符号和单位统一、事件与事务可执行、没有重复柔顺或重复失效、可由 C 层安全反复调用的正式阵列单元模型。不得只拼接、压缩或摘要 B_MODULE_CONTEXT 的 50 个历史章节。

与本轮直接相关的固定工程背景包括：

- 墙面名义平面为全局 X-Y，正 Z 从墙面指向背板和自由空间。
- 单元局部正 x 从头部指向根部，也是搜索和拖拽方向；局部 y 与正 Z 组成右手系。
- 单元背板只允许沿局部 x 和全局 Z 平移；当前 B 层认证运动不包含局部 y 平移或背板转动。
- 阵列为规则矩形田字格，`n_x,n_y∈{2,…,6}`，x/y 等中心距为 4、5 或 6 mm；`2×5` 与 `5×2` 不得视为等价。
- 固定安装角为 50°、60°、70°、80°；线性梯度为根部到头部 `80°→50°` 或 `80°→60°`，并按 `L_r sin(alpha_r)=4 sin(80°)` 保证未加载球心共面。
- 针径为 0.6 或 0.8 mm，针尖半径为 50 或 100 μm；梯度阵列启用针体弯曲时必须使用各排实际露出长度。
- `needle_bending` 与安装模式是正交开关；针梁、局部接触和轴向弹簧是不同物理柔顺，不得重复计入。
- 独立轴向弹簧刚度范围为 100–2000 N/m，无预压、只压缩、不承拉，压缩范围为 0–4 mm，到 4 mm 后进入刚性硬限位；正式刚性安装不得用“大有限刚度”伪装。
- 每单元法向主动推力范围为 0.5–2 N；恒定的是执行器主动推力，不是实际接触合力，更不是平均到每根针的载荷。
- 拖拽速度为 1 mm/s，只用于位移—时间映射；准静态连续行程上限为 100 mm，释放后路径和时间不得重置。
- 一次连续仿真内保留共享轻量不可逆损伤；新独立试验或新表面样本从无损状态开始；原始表面几何不被损伤层重写。
- 首版排除显式裂纹扩展、碎屑/颗粒动力学、连续切削有限元、地形重网格化、针尖磨损、框架/导轨柔性、惯性动力学和大角度运动。
- 表面统计、摩擦、接触柔顺、材料强度与损伤、高碳钢参数、制造误差、弹簧/主动推力离散点、数值容差、能力特征阈值和最终综合评分仍是参数、模型选择或未决项。

## 2. 本轮唯一目标

输出正式 B 阵列单元算子及冻结的 B→C 公共合同，使同一个 B 模型能够：

1. 在独立阵列单元验证模式中，按给定的每单元 `P_z`、局部正 x 路径和最大 100 mm 行程，执行连续共同平衡、事件定位、级联、再挂接和接受历史；
2. 在 C 层嵌入模式中，读取不可变的已接受单元状态和共享损伤快照，接受 C 给定的单元试探运动/控制参数，返回 contact-only 单元 wrench、主动推力与控制/约束量的分栏结果、状态、事件、切线/割线/graph、能力与剩余行程、试探损伤和质量诊断，而不在 C 的全局平衡接受前永久提交任何针级或单元历史；
3. 保持 B1 几何/拓扑、B2 共同平衡、B3 事件事务的单向依赖，不让 C 重新实现 A 单刺机理或 B 阵列载荷共享；
4. 把 C 所需的单元合力/力矩、声明参考点或作用点、局部响应、内部状态、活动针、剩余搜索/弹簧行程、能力边界、失效/恢复事件和完整针级回调条件冻结为稳定公共语义；
5. 对当前 B 运动子空间不能认证的 C 请求显式返回不支持或需扩展状态，尤其不得把整爪 `rocking=on` 引起的真实单元转动静默投影成只有 x/z 平移的 B 请求。

必须建立清楚的调用分层。可以采用现有术语，也可以给出更准确的规范名称，但语义必须唯一：

- intrinsic/embedded array-unit trial：给定不可变接受快照和试探输入，完成 B1+B2+B3 所需的全阵列无副作用试探与内部重平衡，返回可由 C 全局装配的结果和 provisional commit intent；这是 C 的唯一公共调用入口。
- standalone continuous unit driver：在 embedded unit trial 外组织单元级路径推进、1 mm/s 时间映射、100 mm 终止和本层原子接受；只用于 B 层验证/实验对比，不是 C 的公共入口。

如果不做这一分层，B3 的单元内部提交会在 C 的四单元 Newton、事件减步或整体平衡尚未接受时污染历史，B→C 合同即不合格。

### 5.4 唯一方程、约束和集合值装配

把正式 B 单元问题写成可实现的残量、互补和 graph 块，至少包含：

- B1 规则格点、角度/长度、基座反算、共同运动和配置闭合；

- 给定 `u_x,P_z` 时，以 `u_z` 为外层未知量，唯一分支使用
  `r_z=E_Z^T sum_i F_i-P_z=0`，集合值刚性分支使用 `P_z in N_U(u_x,u_z)`；
- 开放但仍可向墙搜索、接触零载、承载、粘滑、弹簧内部/硬限位、材料/针体事件、退化和不可装配响应的正交集合；
- 逐针 wrench 与力矩向共同参考点运输、单元 graph/秩/零空间、原始切线和恒 `P_z` 平衡路径凝聚切线；
- 刚性安装的精确集合值分支和独立弹簧的 `0<=delta_s<=4 mm`、无拉力、硬限位分支；
- 事件后一侧、损伤协调或级联后必须重新调用全部配置针并重新求共同 `u_z`；
- 失效前后法向差量在恒 `P_z` 下闭合，但切向反力和力矩允许变化；
- 物理无解、物理失稳、不可恢复脱附、模型/参数不可用、几何不足、事务错误和数值不收敛的不同证明边界。

### 5.5 重复柔顺、载荷、失效和损伤审计

给出“物理位置—所有者—状态量—力/位移—能量—开关—不得重复项”矩阵，至少覆盖：

必须明确：

### 5.6 规范状态机、事件依赖和并发竞争

把 B1/B2/B3 的重复枚举整理为：

1. 一个互斥的单元主状态；
2. 每针几何/承载子状态；
3. 每支持粘滑/迁移子状态；
4. 每针弹簧/梁结构子状态；
5. 每针材料/针体强度子状态；
6. 单元平衡、唯一性、质量和事务子状态；
7. 瞬时事件组、级联组和提交记录。

对每个状态写明进入条件、保持不变量、允许转移、退出事件、可恢复性和历史提交。必须保留 raw 连续量，不能用状态标签替代力、间隙、裕度、行程或 graph。

其余物理事件不得用武断总 if/else 覆盖。必须在共同位置形成规范同时事件组，并按依赖偏序重算：

接触支持/几何分支 → 粘滑/迁移与弹簧分支 → 当前力状态下材料/针体容量 → 共享损伤协调 → 全阵列共同平衡与容量复核。
